1. 基本介绍

通常方法参数有两种类型：

  • 基本数据类型（数字、布尔值）
  • 对象引用

上面两种类型代表着两种值传递类型，便是值传递和引用传递。

2. 基本数据类型作为参数传递

基本数据类型作为参数传递时都是传递值的拷贝。 指的是方法调用中，传递的是值的拷贝，无论怎么改变这个拷贝，原值是不会改变的。

也即是说实参把它的值传递给形参，对形参的改变不会影响实参的值。
