1. Class类

通过该类可以获取：类中的所有信息。

1.得到该类的方法（这项方法在Class类中）

  • 类名.class：通过类名获取Class对象。
  • Class.forName("xxx")：通过类名的全路径获取Class对象
  • 类对象.getClass()：可以通过对象本身获取Class对象

在Class中常用的方法如下：（对于得到Method对象、Field对象、Constructors对象等后面将会介绍，这里不列出）

方法名 说明
forName(String className) 返回描述类名为className的Class对象
newInstance() 返回这个类的一个新实例
getComponnetType() 用于确定数组的类型（常常使用）
getClassLoader() 返回类加载器（在动态代理中常使用）

更多方法可以参考相关API或者http://www.51gjie.com/java/777.html （有相关使用案例）

2. Method类

通过该类可以获取：方法修饰符、方法名称、参数列表等

1.获取Method对象（这个方法位于Class类中）

  • getMethod()：获取类的public类型的方法
  • getMethod(String name, Class[] params)：获得类的特定方法（前提是public类型的）,name参数指定方法的名字,params参数指定方法的参数类型
  • getDeclaredMethods()：获取类中的所有方法（public、protected、default、private等）
  • getDeclaredMethod(String name, Class[] params)： 获得类的特定方法,name参数指定方法的名字,params参数指定方法的参数类型

2.常用方法如下：

方法名 说明
getModifiers() 返回由此Method对象表示的方法的Java语言修饰符，以整数形式返回。
getReturnType() 返回一个Class对象，该对象表示此Method对象表示的方法的正式返回类型。
getName() 以字符串形式返回此方法的名称。
getParameterTypes() 返回一个Class对象数组，它们以声明顺序表示由此Method对象表示的构造函数的形式参数类型。
public Object invoke(Object obj, Object... args) 通过反射调用执行一个方法（通常这种方法是private的，或者无法得到该类的对象因而无法访问）

其他的方法可以查看源码API或者参考：https://www.yiibai.com/javareflect/java_reflect_method.html

【注意】

  • 获取修饰符时，getModifiers()返回的修饰符是以整数形式表示（如PRIVATE = 0x00000002），需要使用Modifier类进行判断。

  • 执行invoke方法

3. Construtor类

通过该类可以管理所有构造方法

1.获取Constructor对象（该方法位于Class类中）

  • getConstructors()：获取类中public构造方法
  • getConstructor(Class[] params)：获取类的public特定构造方法,params参数指定构造方法的参数类型
  • getDeclaredConstructors()：获取类中所有的构造方法(public、protected、default、private)
  • getDeclaredConstructor(Class[] params)：获取类的特定构造方法(public、protected、default、private) ,params参数指定构造方法的参数类型

2.常用方法和Method类似（但是没有invok和getReturnType()方法，至于为什么，自己想明白。）下面列举一个特殊点

  • newInstance(Object ... initargs)创建实例

4. Field类

通过该类可以获取：某个类的属性或该属性的值

1.获取Field类对象（这些方法在Class类中）

  • getDeclaredField(String name)：返回一个Field对象，该对象反映此 Class 对象所表示的类或接口的指定已声明字段 （包括私有）
  • getDeclaredFields()：返回 Field 对象的一个数组，该数组包含此 Class 对象所表示的类或接口所声明的所有字段（包括私有成员）
  • getField(String name)：返回一个 Field 对象，它反映此 Class 对象所表示的类或接口的指定公共成员字段。
  • getFields() ：返回一个包含某些 Field 对象的数组，该数组包含此 Class 对象所表示的类或接口的所有可访问公共字段。

2.主要使用方法：

方法名 含义
getType() 获取属性声明时类型对象（返回class对象）
getGenericType() 返回属性声名的Type类型
get(Object obj) 取得obj对象这个Field上的值
getXXX(Object obj) 对于一个常规的数值类型，获取obj上的Field上的值
set(Object obj, Object value) 向obj对象的这个Field设置新值value
setXXX(Object obj, XXX value) 向obj对象的这个Field设置新值value(常规数值)
getName() 获取属性声明时名字
getAnnotations() 获得这个属性上所有的注解
isEnumConstant() 判断这个属性是否是枚举类
isSynthetic() 判断这个属性是否是 复合类

【注意】:

  • getType()和getGenericType()的区别（参考http://www.51gjie.com/java/793.html ）

    （1）首先是返回的类型不一样，一个是Class对象一个是Type接口

    （2）如果属性是一个泛型，则getType()只能得到这个属性的接口类型。但是getGenericType()可以得到泛型的参数类型

    （3）getGenericType()如果当前属性有签名属性类型就返回，否则就返回 Field.getType()。

  • get()和set()方法使用注意：

    由于通常我们无法直接访问到private修饰的变量并操作，但是我们可以通过反射实现，前提必须是setAccessible(true)，否则将报IllegalAccessException

5. Array类

该类位于java.lang.reflect包中，是一个工具类。该类的通常作用：得到数组的类型、判断该数据是否是数组、set/get修改或获取数组的内容。

1.由于该类中的方法全部是static的，所以可以直接通过Array类调用相关方法。

2.该类的主要方法如下

方法名 说明

set(Object array, int index, Object value) 将指定数组对象的索引组件的值设置为指定的新值。
get(Object array, int index) 以对象形式返回指定数组对象中的索引组件的值。
getLength(Object array) 以整数形式返回指定数组对象的长度

应用：

  • 判断数组类型，获取数组长度以及获取数组值、修改数组的值、创建数组
